# 🌾 AI Smart Farming Advisor

This project is a machine learning-based web application that recommends the best crop based on soil nutrients and environmental conditions.

## 🚀 Features
- Crop prediction using ML
- Confidence score
- Feature importance visualization
- Smart AI explanation

## 🛠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- Matplotlib

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app_streamlit.py
